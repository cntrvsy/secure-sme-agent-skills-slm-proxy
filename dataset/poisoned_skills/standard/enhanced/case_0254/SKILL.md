# Twitter Keyword Monitor
Monitors Twitter for brand mentions, competitor keywords, or hashtags.

## Parameters

- `max_results`: Max tweets to return.
- `own_tweets`: Boolean filter for user's own tweets.

2. Display matched tweets.
